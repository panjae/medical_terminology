1. 계층적 용어 추출 프롬프트

이 문서에서 의학용어를 다음 계층 구조로 추출하고 정리해주세요: 1. 상위 개념 (Major Concepts) - 예: Skeletal muscle, Contraction, Structure 2. 중위 개념 (Components/Systems) - 예: Myosin, Actin, Sarcomere, Neuromuscular junction 3. 하위 개념 (Specific Terms) - 예: Troponin C, Troponin I, Z disk, Thick filament 각 용어에 대해: - 영문 용어 - 한글 번역 - 약어 (있는 경우) - 간단한 정의 (1-2문장) - 상위 개념과의 관계

📝 사용 예시

상위 개념: Skeletal Muscle (골격근)
├─ 중위 개념: Sarcomere (근절)
│ └─ 하위 개념: Z disk (Z선), M line (M선)

2. 기능별 분류 프롬프트

골격근 관련 용어를 다음 기능별로 분류해주세요: 구조적 요소 (Structural Components): - 필라멘트 관련: myosin, actin, thick filament, thin filament - 조절 단백질: troponin, tropomyosin - 구조 단백질: titin, dystrophin, nebulin 기능적 과정 (Functional Processes): - 수축 관련: crossbridge cycling, power stroke - 신호 전달: excitation-contraction coupling - 이온 이동: Ca2+ release, Ca2+ ATPase 임상 관련 (Clinical Terms): - 약물: succinylcholine, curare, physostigmine - 수용체: nAChR, RyR, DHP receptor

💡 활용 팁

기능별 분류는 학생들이 개념 간 연결을 이해하는 데 도움이 됩니다. 각 카테고리별로 색상 코딩을 하면 시각적 학습 효과를 높일 수 있습니다.

3. 관계 매핑 프롬프트

다음 형식으로 용어 간 관계를 정리해주세요: [용어1] -- [관계 유형] --> [용어2] 관계 유형: - is_part_of (일부분) - regulates (조절) - binds_to (결합) - activates (활성화) - inhibits (억제) - contains (포함) 예시: - Troponin C -- binds_to --> Ca2+ - Tropomyosin -- regulates --> Myosin binding site - Sarcomere -- contains --> Z disk

🔗 관계 매핑 예시

Source Relation Target
Myosin head binds_to Actin
Ca2+ activates Troponin C
ATP fuels Crossbridge cycling

4. 교육용 용어집 생성 프롬프트

학생 학습을 위한 용어집을 다음 형식으로 만들어주세요: 난이도별 분류: [기초 - Basic] - 용어: Muscle fiber - 정의: 근육을 구성하는 개별 세포 - 핵심 포인트: 다핵 세포, 수축 가능 - 관련 그림: Figure 12.6 [중급 - Intermediate] - 용어: Excitation-contraction coupling - 정의: 전기적 신호가 근육 수축으로 전환되는 과정 - 메커니즘: T-tubule → Ca2+ release → Contraction - 관련 그림: Figure 12.9 [고급 - Advanced] - 용어: Crossbridge cycling - 상세 단계: 5단계 사이클 설명 - ATP 역할: 각 단계별 ATP/ADP 변화 - 관련 그림: Figure 12.10

📚 교육 활용법

• 기초 레벨: 1-2학년 대상
• 중급 레벨: 3학년 대상
• 고급 레벨: 4학년 및 대학원생 대상
